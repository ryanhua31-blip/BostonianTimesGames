# Bostonian Times Games

A static newspaper-style games page built with plain HTML, CSS, and JavaScript.

## Included

- A Bostonian Times games landing page inspired by a newspaper puzzle hub
- A playable `Word Links` game on a 15 x 15 grid
- Starter words, move validation, scoring, move history, and replay support

## Open locally

Open `index.html` in a browser.
